# مشروع الطريق البحري - الهيكل التنظيمي

تطبيق مبني باستخدام Flutter يعرض الهيكل التنظيمي المقترح لمشروع تنفيذ الطريق البحري. يوفر التطبيق واجهة مستخدم متجاوبة تعمل بشكل ممتاز على الهواتف المحمولة (عبر قوائم قابلة للتوسيع) وعلى الأجهزة اللوحية والمكتبية (عبر رسم تخطيطي شجري).

## الميزات
* عرض هرمي واضح للمناصب بدءاً من مدير المشروع.
* واجهة متجاوبة (Responsive Design).
* تصميم حديث مدعوم بـ Material 3.
* دعم اتجاه النص من اليمين لليسار (RTL) للغة العربية.

## الإدارات الرئيسية
* الإدارة الهندسية
* إدارة العمليات
* إدارة الجودة والسلامة
* الشؤون الإدارية والمالية

## التشغيل
للتشغيل محلياً:
```bash
flutter pub get
flutter run
```
